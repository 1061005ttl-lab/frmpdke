📋 覆蓋率看板 — 怎麼看這個畫面？

【這個看板在追蹤什麼？】追蹤各店家的廣宣物料覆蓋狀況。每一筆資料代表一家店，狀態只有三種：已知有（業務已完成貼附）、已知無（業務確認無法執行，並填寫原因）、未填寫（業務尚未回報）。

【卡片上的數字怎麼看？】

每張卡片代表一個單位（營業處 / 營業所 / 業務人員）。卡片內的數字方塊含義如下：

⬜ 總家數：這個單位負責的店家總數
🟥 未填寫：業務尚未回報的店家數（紅色代表需要催填）
🟩 已貼開瓶 / 已貼黃董（B群）：已成功貼附對應廣宣物料的店家數
🟥 未商化（C群）：業務已確認無法貼附的店家數（含原因說明）

數字下方的 % = 該狀態家數 ÷ 總家數，讓你快速掌握覆蓋率進度。

將滑鼠移到數字上（手機長按），可展開 提示工具，看到細項拆分（如：開瓶海報-店外 3 家、開瓶冰貼 5 家），以及在該群組內的佔比。

【上方篩選列怎麼用？】

🏢 組織 / 通路：切換兩種視角。組織視角依「營業處 → 營業所 → 業務人員」逐層鑽取；通路視角依「通路別 → 營業處 → 營業所」鑽取。
等級 ▶：展開後可選擇只看 A 級或 B 級店家，點「全選」回到全部。
全覽 / 北部 / 桃園…：點擊直接跳到對應層級，點「全覽」回到最上層。
已貼開瓶 / 已貼黃董 / 未商化 / 未填寫（燈號）：點擊可關閉某個狀態類別，讓卡片統計與明細只顯示你關心的那幾類。

【點進明細 Dialog 看什麼？】

點卡片右上角的 📋 明細，或直接點卡片本身（最末層），會開啟店家明細表。

• 標題列顯示目前的篩選條件和各狀態家數及佔比
• 表格欄位依群組上底色：綠色 = B1 群（已貼開瓶）、藍色 = B2 群（已貼黃董）、紅色 = C 群（未商化）
• ✅ = 該欄位有值（已貼附）；✖ = 該欄位有值（未商化原因）；— = 無此狀態
• 利用燈號篩選器可只顯示特定群組後再截圖，截圖只含需要的資料

【卡片右上角的小按鈕是什麼？】

💬 複製 LINE 訊息：產生條列式文字，分已貼 / 未商化 / 未填寫三大區塊，貼到 LINE 群組直接發送
📥 下載 CSV：下載該單位的店家資料，檔名含單位名稱與時間戳，用 Excel 開啟可進一步分析
📸 截圖：自動開啟明細並截圖，方便存檔或分享

Dialog 內也有相同的三個按鈕，操作結果相同。

【資料更新時間】看板右上角顯示最後同步時間（台灣時間），每次開啟頁面會自動從 Google 試算表讀取最新資料。
