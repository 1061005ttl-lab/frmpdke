📋 覆蓋率型看板 — 操作說明

【這是什麼看板？】 這是一個從 Google 試算表即時讀取資料的覆蓋率追蹤看板。每一筆資料代表一家店，狀態分三類： 已知有（B 群，綠色）、已知無（C 群，紅色）、未填寫（紅色警示）。 看板的目的是讓管理者一眼掌握各層級的覆蓋進度，並快速輸出資料給現場業務。

【數字卡片怎麼讀？】 每張卡片代表一個層級單位。卡片內有多個數字方塊：

總家數：這個單位負責的全部店家數
未填寫（紅）：尚未回報的店家數，% = 未填寫 ÷ 總家數
B 群各項（綠）：已知有的店家數，可有多個群組（如已貼 A 物料、已貼 B 物料）
C 群各項（紅）：已知無的店家數，可有多個群組（如原因 X、原因 Y）
所有 % 以當前可見總家數為分母，隨燈號篩選動態更新

👉 將滑鼠移到數字上（手機長按），展開細項提示工具，看到每個子欄位的家數與群組內佔比。

【頂部工具列】

← 返回：回到專案選單
專案選單（下拉）：切換不同覆蓋率專案，每個專案讀取獨立的試算表
☰ 選單（手機）：收納視角切換與所有輸出功能，點擊展開
🏢 組織 / 🏪 通路 / 🏷️ 等級：切換鑽取視角。組織視角依「營業處→所→人員」；通路視角依「通路別→處→所」；等級視角依「店家等級→營業處→所→人員」
🖼️ 照片牆：長官流水線視角，把目前範圍內所有拜訪紀錄攤平成一份時間軸，只列出有拍照的紀錄，新到舊排序，一眼看完業務員今天去了哪、拍了什麼
同步時間：顯示資料最後更新時間（台灣時間）
💬 複製當前頁面的 LINE 完整版訊息（含全部店家分組明細）
✉️ 複製 LINE 精簡版訊息（僅統計摘要，不列店家名）
📥 下載當前頁面資料的 CSV 檔（檔名含單位與時間戳）
📸 截圖目前可見的整個頁面（含 topbar / 燈號 / 卡片），直接複製到剪貼簿

【篩選列（toolbar 下方）】

燈號（已知有 / 已知無 / 未填寫）：點擊切換 ON/OFF，關掉的類別從卡片統計與明細中移除，% 跟著重算。可單獨只留某一類做截圖
等級 ▶/◀：展開後選擇店家等級篩選（如 A 級、B 級），點「全選」回到全部
全覽 / 各區域名稱：點擊直接跳到該層級，點「全覽」回最上層。點卡片可往下鑽一層

【店家明細 Dialog】

點卡片右上角 📋 明細 或直接點擊最末層卡片，開啟該單位的店家逐筆列表。

標題列顯示各狀態家數與佔比，隨燈號篩選動態更新
表格欄位依群組上色：綠 = B1 群　藍 = B2 群　紅 = C 群（顏色數量依專案設定而異）
✅ = 有值（已貼附）　✖ = 有值（已知無原因）　— = 無此狀態
利用燈號可只顯示需要的類別，再操作 LINE/CSV，輸出的內容只含當前可見資料
點任一店家列可再展開該店的完整拜訪時間軸（同一店可能有多筆歷史紀錄，預設只展開最新一筆，其餘收合，點擊即可查閱）

【卡片與 Dialog 的輸出按鈕】

💬 LINE 完整：條列式訊息，分「已知有（各 B 群）→ 已知無（各 C 群）→ 未填寫」三大區塊，每家店占一行，尾端附各欄統計。貼到 LINE 群組直接發送
✉️ LINE 精簡（toolbar 專屬）：只輸出統計摘要數字，不列店家名稱，適合快速回報進度
📥 CSV：下載含狀態欄的完整資料，檔名格式「單位_YYYYMMDD_HHMM.csv」，以 Excel 開啟可進一步分析
📸 截圖：直接複製圖片到剪貼簿，貼上 LINE／Email 即可，不會另外下載檔案
