01 這是什麼看板

這是一個即時讀取資料表的覆蓋率追蹤看板。每一筆資料代表一家店，狀態分三類：已知有（B 群，綠色）、已知無（C 群，紅色）、未填寫（紅色警示）。

看板目的是讓管理者一眼掌握各層級的覆蓋進度，並快速輸出資料給現場業務。

02 數字卡片怎麼讀

每張卡片代表一個層級單位，卡片內有多個數字方塊：

  • 總家數：這個單位負責的全部店家數
  • 未填寫（紅）：尚未回報的店家數，% = 未填寫 ÷ 總家數
  • B 群各項（綠）：已知有的店家數，可能有多組（如已貼 A 物料、已貼 B 物料）
  • C 群各項（紅）：已知無的店家數，可能有多組（如原因 X、原因 Y）

所有百分比都以「當前可見總家數」為分母，隨燈號篩選動態更新。

👉 滑鼠移到數字上（手機請長按），會展開細項提示，看到每個子欄位的家數與群組內佔比。

03 頂部工具列

← 返回：回到專案選單。專案選單：切換不同覆蓋率專案，各專案資料互相獨立。☰ 選單（手機）：收納下面三組功能，點擊展開。

🏢 組織 / 🏪 通路 / 🏷️ 等級：切換整個頁面的鑽取視角——組織視角依「營業處→所→人員」；通路視角依「通路別→處→所」；等級視角依「店家等級→營業處→所→人員」。

同步時間：顯示資料最後更新時間（台灣時間）。

不管目前鑽到哪一層，這些按鈕永遠以「當前可見整體範圍」開啟彙整看板。

這組是針對「目前頁面」燈號篩選後的結果輸出，跟彙整看板內部輸出是分開的兩件事。

04 篩選列

  • 燈號（已知有 / 已知無 / 未填寫）：點擊切換 ON/OFF，關掉的類別會從卡片統計與明細中移除，可以只留某一類方便截圖。
  • 等級 ▶/◀：展開後選擇店家等級篩選，點「全選」回到全部等級。
  • 全覽 / 各區域名稱：點擊直接跳到該層級，點「全覽」回最上層；點卡片則往下鑽一層。

05 彙整看板 Dialog（6 分頁）

點工具列「彙整看板」任一按鈕開啟，範圍固定是「目前整體可見範圍」，跟目前鑽到哪一層無關；分頁間切換不會清空已勾選的細項匯出項目，但每次重新開啟彙整看板都會重置。

分頁看什麼
👤 人員 / 🏢 單位 / 🏪 店家三種不同分組角度的彙總表，可再往下展開明細
🖼️ 照片牆只看有拍照的拜訪紀錄，時間新到舊排列，附店名／單位／人員
🔍 逐題彙整左側選一個大題底下的某個選項，右側列出範圍內所有填了這個選項的店家與文字回覆
🧩 細項匯出勾選一個或多個子欄位（可跨大題勾），即時列出符合的店家，依「單位→人員」分組，可輸出 LINE 或 CSV

對話框右上角的 📥 CSV / 📸 截圖 只會輸出目前分頁的內容。

06 店家明細 Dialog

點卡片右上角 📋 明細 或直接點擊最末層卡片，開啟該單位的店家逐筆列表。

  • 標題列顯示各狀態家數與佔比，隨燈號篩選動態更新。
  • 表格欄位依群組上色：綠 = B1 群　藍 = B2 群　紅 = C 群（顏色數量依專案設定而異）。
  • ✅ = 有值（已貼附）　✖ = 有值（已知無原因）　— = 無此狀態。
  • 點任一店家列可再展開該店的完整拜訪時間軸（同一店可能有多筆歷史紀錄，預設只展開最新一筆）。
  • 工具列 🧩 細項匯出 按鈕：直接關閉本明細、開啟彙整看板並跳到「細項匯出」分頁——範圍是目前整體彙整範圍，不是只有這份明細。

07 輸出按鈕

  • 💬 LINE 完整：條列式訊息，分「已知有（各 B 群）→ 已知無（各 C 群）→ 未填寫」三大區塊，每家店占一行，尾端附各欄統計，可直接貼到 LINE 群組。
  • ✉️ LINE 精簡：只輸出統計摘要數字，不列店家名稱，適合快速回報進度。
  • 📥 CSV：下載含狀態欄的完整資料，檔名格式「單位_YYYYMMDD_HHMM.csv」，用 Excel 開啟即可分析。
  • 📸 截圖：直接複製圖片到剪貼簿，貼上 LINE／Email 即可，不會另外下載檔案。

08 常見問題

Q：截圖按了沒反應？
A：請確認瀏覽器允許存取剪貼簿權限，允許後再試一次。

Q：切了燈號篩選後，百分比怎麼跟之前不一樣？
A：所有百分比都是以「目前可見總家數」為分母，關掉某個燈號類別會讓分母跟著變動，這是正常設計，不是錯誤。

Q：手機上長按數字沒有跳出提示？
A：請確認手指按住不放約 1 秒（避免跟單擊卡片鑽層的動作衝突），放開後提示會再停留一下才消失。
